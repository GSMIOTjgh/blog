# 프로젝트 구성하기

`create astro`를 통해서 기본 프로젝트를 구성할 수 있습니다. ex. `bun create astro`
프로젝트 이름 정하고 `Include sample files`를 통해서 샘플을 만들어봅시다.

생성된 파일을 보면 크게 **public**과 **src**를 볼 수 있어요.

src는 웹사이트를 구성하는 요소들이 있는 파일이고,
public은 favicon.ico와 같이 정적인 요소들이 들어가요.

src를 구성하는 요소들은 다음과 같아요.

1. components
2. layouts
3. pages

> components는 react 써보셨으면 알 것 같지만! 작은 기능적 단위를 나타내는데 버튼이나 헤더 단위로 쪼갠 것이라고 생각하면 쉬울 것 같아요.
> layouts는 기본적인 UI를 정의하고, pages는 말 그대로 `/`, `/about`과 같은 페이지를 나타냅니다.
