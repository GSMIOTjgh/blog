# @Singular

Singular 어노테이션을 사용하면 모든 원소(List(Long) score)를 한번에 넘기지 않고 개별(Long score)로 넘길 수 있습니다.

```java
@Builder
public class Student {
  private UUID id;
  private String email;
  private String password;
  @Singular
  private List<Long> scores;
}
```

이러한 엔티티가 존재한다고 하면 다음과 같이 builder를 사용할 수 있습니다.

```java
Student student = Student.builder()
  .id(UUID.randomUUid())
  .email("me@xolving.com")
  .password("iHaveToDo123")
  .score(100)
  .score(90)
  .score(96)
```

100, 90, 96의 원소를 포함한 scores를 가진 student 인스턴스가 만들어집니다.
